<h2 align="center">üìñ Project Overview</h2>
<p>
    <strong>Traces of Twins</strong> is a 3D story-driven mystery game optimized for PC and mobile platforms. The player takes on the role of a detective, solving a complex case involving missing twins through exploration, evidence collection, and logic puzzles.
</p>
<p>
    As the <strong>Team Lead</strong> and <strong>Lead Developer</strong>, I directed the entire creative and technical process; from designing minigame mechanics to engineering the core systems that bring the story to life.
</p>
<h2 align="center">üéÆ Key Features</h2>
<ul>
    <li><strong>Immersive Investigation:</strong> A complete detective loop including DNA collection, fingerprint analysis, and evidence placement.</li>
    <li><strong>Diverse Minigames:</strong> Over 10 unique minigames (Lockpicking, Wiring, Hacking, Simon Says) designed for touch and mouse interaction.</li>
    <li><strong>Mobile & PC Optimization:</strong> High-fidelity 3D environments optimized using <strong>Occlusion Culling</strong> and <strong>Canvas-based Interaction</strong>.</li>
    <li><strong>Narrative Systems:</strong> A dynamic dialogue and quest system that guides the player through a linear mystery.</li>
</ul>
<h2 align="center">‚öôÔ∏è Technical Engineering & Design Leadership</h2>
<p>I was responsible for the <strong>Entire Unity Implementation</strong> and <strong>Game Design Direction</strong>, ensuring art assets served the gameplay mechanics efficiently.</p>
<h3>üèóÔ∏è System Architecture & Optimization</h3>
<ul>
    <li><strong>Canvas-Based Minigames:</strong> To maximize performance, I architected all minigames to run entirely on the <strong>Unity UI (Canvas)</strong> system rather than using 3D physics. This significantly reduced draw calls and physics overhead.</li>
    <li><strong>Event-Driven Design:</strong> Architected the core game loop using <strong>C# Events</strong> to decouple the UI, Inventory, and Quest systems.</li>
    <li><strong>Interaction Framework:</strong> Developed a modular interaction system allowing seamless switching between 3D exploration and 2D puzzle interfaces.</li>
</ul>
<h3>‚úçÔ∏è Narrative & Level Design</h3>
<ul>
    <li><strong>Narrative Direction:</strong> While the core script was written by an external writer, I designed the <strong>Narrative Flow</strong>, wrote all <strong>In-Game Dialogues</strong>, and implemented the branching logic.</li>
    <li><strong>Level Design:</strong> Built the entire world layout in Unity, ensuring the environment supported the investigation loop.</li>
    <li><strong>Design Leadership:</strong> Defined the mechanics and logic for all minigames, directing the art team on the visual requirements and UX flow.</li>
</ul>
<h3>üé® Technical Art & Asset Creation</h3>
<ul>
    <li><strong>Environment Art:</strong> Personally <strong>modeled and textured the Police Station</strong> environment using Blender, ensuring it fit the performance budget.</li>
    <li><strong>UI Assets:</strong> Created high-quality icons for collectible items by setting up <strong>render scenes in Blender</strong>, bridging the gap between 3D assets and 2D UI.</li>
    <li><strong>Lighting & Atmosphere:</strong> Baked lighting maps and configured real-time shadows to achieve a "Noir" atmosphere.</li>
    <li><strong>Animation Rigging:</strong> Handled NPC rigging and animation blending using <strong>Mixamo</strong> and Unity's Animator State Machines.</li>
</ul>
